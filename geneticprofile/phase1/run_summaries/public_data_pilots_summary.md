# Public Data Pilots Summary

The public-data pilots were used as context calibrators and possible source-object tests.

## Processed As Useful Context

- Roquetes medieval Jewish Catalonia (`PRJEB107193`): useful chr9 broad/core context, especially in denser rows; replacement-source status absent.
- Amvrakia (`PRJNA1143893`): processable western-Greek/Hellenistic-Roman context; neutral to near-null under Ashkenazi controls.
- Roca Vecchia (`PRJEB93761`): South-Italy/Aegean Bronze Age context; whole-genome neutral, broad chr9 echo control-sensitive.
- Northeast Iberian Iron Age (`PRJEB111574` / `ERP192183`): bounded Western Mediterranean context; source-object support absent.
- Nahal Yarmuth 38 (`PRJEB83268`): low-power deep-prehistoric southern-Levant context; WGS rows sparse and targeted-capture rows mitochondrial-only.

## Data Gaps

The major ancient specificity gap remains dense late southern-Levant Jewish-context genome-wide data from Judea, Galilee, Jerusalem, or a clearly Jewish southern-Levant context around roughly 300 BCE-300 CE.

The major Roma/Romani upgrade is matched Eastern/Central European Roma data. `EGAD50000001103` is the best identified dataset, with managed-access Czech Roma and Czech non-Roma Human Origins array samples.

## Public Package Rule

This package lists accessions and summarizes outcomes. It excludes downloaded reads, transformed reference genotypes, and local intermediate files.
