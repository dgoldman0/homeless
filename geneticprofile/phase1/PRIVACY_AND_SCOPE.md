# Privacy And Scope

This package is built for public upload. It preserves the scientific audit trail while keeping the analyzed genome and living-person context private.

## Public-Safe Contents

- Aggregate evidence tables.
- Window coordinates and status labels.
- Named public comparator samples from published or public panels.
- Public accession IDs and dataset descriptions.
- Paper figures whose source data are aggregate or schematic.
- Method notes and guardrails.
- Claims-index rows that connect results to evidence classes.

## Excluded Material

- Raw genotype, sequencing, and imputed files.
- Per-marker genotype rows for the study target.
- Phased target haplotypes.
- IBD match lists involving private individuals.
- Living-person identifiers, account identifiers, kit IDs, emails, and private family-record files.
- Third-party PDFs and reference datasets whose licenses or terms are handled at the source site.
- Private-like Y SNP detail beyond the public-safe branch placement.

## Use Model

Readers can audit whether the paper's claims follow from the evidence hierarchy. They can also identify which future public datasets would sharpen a result. The package intentionally leaves private target inputs outside the public repository.
