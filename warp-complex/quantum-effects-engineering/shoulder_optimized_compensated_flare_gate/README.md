# Shoulder-Optimized Compensated Flare-Gated Radial Stretch

This bundle contains a reduced geometry screen for the shoulder subsystem of the compensated flare-gated radial-stretch design.

## Contents

- `memo/shoulder_optimization_memo.md` — design memo and interpretation.
- `scripts/run_shoulder_optimization_screen.py` — reproducible reduced screen.
- `data/shoulder_geometry_broad_sweep.csv` — broad comparison of shoulder R, shoulder N, combined R/N, and explicit source branches.
- `data/balanced_source_shoulder_sweep.csv` — focused sweep for balanced support-plus-shoulder compensation.
- `data/extracts.json` — key results and selected reference case.
- `manifest.json` — file hashes and bundle metadata.

The screen uses effective-source diagnostics from a prescribed spherical metric. It is a geometry-engineering screen, not a quantum-state construction.
