# Adiabatic radial process bundle

This bundle contains a standalone memo and associated evaluation artifacts for the adiabatic radial-stretch setup/hold/reset protocol.

## Contents

- `adiabatic_radial_process_memo.md` — standalone memo.
- `scripts/run_adiabatic_radial_protocol_eval.py` — generator script copied from the local evaluation.
- `data/adiabatic_radial_protocol_case_summary.csv` — full case summary.
- `data/adiabatic_radial_protocol_scaling.csv` — log-log scaling estimates.
- `data/adiabatic_radial_protocol_extract_table.csv` — selected cases.
- `data/adiabatic_radial_protocol_time_digest.csv` — compact time histories.
- `data/adiabatic_radial_protocol_extracts.json` — selected summary extracts.
- `figures/` — scaling plots for the representative B0=3, wB=5 minimum-jerk protocol.
- `sources/literature_review_sources.md` — source guide.
- `sources/references.bib` — BibTeX references.
- `manifest.json` — checksums.

## Scope

This is a prescribed-geometry engineering screen. It evaluates whether adiabatic radial proper-length setup/hold/reset behavior has clean dynamic cost scaling before source-realism and quantum-inequality modeling.
