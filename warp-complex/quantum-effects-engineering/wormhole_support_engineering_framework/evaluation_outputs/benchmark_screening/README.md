# Wormhole-support engineering screening benchmarks

This bundle combines the phase-1 benchmark memo and phase-2 pilot benchmark code.

The benchmark goal is to test whether the proposed engineering screening framework reproduces known verdicts from canonical wormhole/QI literature examples. The script encodes reduced proxy cases rather than full numerical reconstructions of every published geometry.

Files:

- `benchmark_screening_memo.md` — benchmark memo and interpretation.
- `run_benchmark_screen.py` — generator for benchmark CSVs.
- `benchmark_model_catalog.csv` — canonical benchmark model catalog.
- `benchmark_gate_summary.csv` — gate sequence assigned to each benchmark.
- `benchmark_screening_table.csv` — compact field-facing benchmark table.
- `references.md` — literature anchors.
- `manifest.json` — file checksums.
