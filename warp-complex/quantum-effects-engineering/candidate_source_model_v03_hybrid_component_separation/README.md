# Candidate Source Model v0.3: Hybrid Component Separation

This bundle records the v0.3 candidate-source model screen for the frozen Reference Geometry v0.3. The main report is:

`report/candidate_source_model_v03_report.md`

The bundle includes the locality-polished component separation screen, the residual-repayment strict-pass variant, context from candidate source model v0.2, source-realism prescreen v0.2, and the frozen geometry v0.3 invariant outputs.

## Main result

The leading source architecture is:

`NMC scalar-like support + support-ring infrastructure repayment + shoulder/matching repayment`.

The NMC-like component carries the core/access/support role during the open interval. The infrastructure component carries residual support-ring and shoulder repayment.
