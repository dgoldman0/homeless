# Lapse and Radial-Metric Freedom Throat Evaluation

This bundle contains the reduced wormhole-only evaluation of lapse and radial-metric freedom for a phase-cycled throat plant.

The main report is:

- `lapse_radial_freedom_throat_report.md`

Primary scripts:

- `run_lapse_radial_freedom_eval.py`
- `run_proper_length_optimized_sweep.py`

Primary outputs:

- `lapse_radial_case_summary.csv`
- `lapse_radial_parameter_sweep.csv`
- `lapse_radial_time_digest.csv`
- `lapse_radial_result_extracts.json`
- `proper_length_optimized_sweep.csv`
- `proper_length_profile_digest.csv`
- `proper_length_sweep_extracts.json`
- `proper_length_sweep_summary.json`

Scope: wormhole/QEE component only. The transport/catch/passenger layer is outside this bundle.
