# Phase-Cycled Throat Support: Audited Reproducible Bundle

This bundle replaces the earlier non-reproducible package. The outputs in this folder were regenerated from the included scripts and verified by SHA-256 checksums.

Run from this directory:

```bash
python stage1_static_wormhole_qi/run_stage1_static_wormhole_qi.py
python stage2_dynamic_throat_qi/run_stage2_dynamic_throat_qi.py
```

The scripts write outputs into their own folders. The model is a reduced QI proxy, not a full semiclassical calculation.
