# Active Rail Phase A Optical/Radiation-Adjacent Screen

This bundle contains a reduced radial Phase A screen for the active-rail model.

## What it estimates

- local packet-frame photon energy factor
- reduced radial characteristic reachability
- radial focusing / magnification proxy
- heuristic flux-risk proxy
- active-branch valid-ray mask
- branch comparison for local photon gain

## What it does not estimate

- full 3D off-axis geodesics
- RSET / semiclassical radiation
- particle pickup
- Geant4/material shielding
- absorbed dose in passengers
