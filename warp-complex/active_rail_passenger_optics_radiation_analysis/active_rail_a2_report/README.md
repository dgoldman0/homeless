## Short result

A2 strongly suggests that the large Phase A radial focusing spike is mostly a one-dimensional radial artifact. Once rays carry impact parameter in a frozen-frame 2D equatorial screen, focusing falls sharply and photon energy shift remains below unity. This does not certify radiation safety, but it keeps the honest viewport branch alive and reinforces the broader pattern that the frozen active-rail architecture continues to survive diverse reduced probes without needing redesign.
