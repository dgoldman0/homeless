# Reproducibility notes

This paper bundle contains compact CSV summaries used by the LaTeX paper and figures. The full exploratory per-slice tensor archives are intentionally excluded from the GitHub-sized paper bundle. The CSV files encode the transition refinement, geometry confirmation, quantum-source proxy screen, pressure robustness, and grid sensitivity values reported in the paper.

Main data files:

- `data/transition_refinement_summary.csv`
- `data/catch_thresholds.csv`
- `data/geometry_confirmation_summary.csv`
- `data/quantum_source_proxy_summary.csv`
- `data/robustness_pressure_summary.csv`
- `data/grid_sensitivity_summary.csv`

The figures under `figures/` were generated from those compact tables.
