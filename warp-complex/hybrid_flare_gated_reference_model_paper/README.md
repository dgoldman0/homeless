# Hybrid Flare-Gated Reduced Reference Model Paper Bundle

This folder contains a standalone LaTeX paper, compiled PDF, reproducibility scripts, data tables, figures, literature access notes, and manifest for the Hybrid Flare-Gated Reduced Reference Model.

## Main outputs

- `latex/main.tex` — standalone paper source.
- `latex/references.bib` — bibliography database.
- `pdf/hybrid_flare_gated_reference_model.pdf` — compiled paper PDF.
- `data/` — CSV/JSON outputs used by the paper.
- `scripts/` — scripts that generated the geometry, source, component-separation, and closure screens.
- `figures/` — figures included in the paper.
- `docs/literature_sources.md` — source access notes for cited literature.
- `docs/hybrid_flare_gated_reduced_reference_model_v1_0_technical_report.md` — preceding technical report.

## Scope

The paper freezes a reduced engineering reference model. It presents a geometry/source architecture, observer-family ledgers, flux-completed anisotropic closure, and sampled-stress physics gate. It supplies a target for quantum-source admissibility work.
